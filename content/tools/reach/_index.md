# Reach

```
NAME:
   reach

USAGE:
   Reach builds modules of Repeatr operations, stages releases of the results, and can commission builds of entire generations of atomic releases from many modules.

COMMANDS:

    emerge
        evaluate a pipeline, logging intermediate results and reporting final exports

    ci
        given a module with one ingest using git, build it once, then build it again each time the git repo updates

    catalog
        catalog subcommands help maintain the release catalog info tree

    help, h
        Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --help, -h     show help
   --version, -v  print the version
```
