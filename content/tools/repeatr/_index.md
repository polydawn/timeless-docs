# Repeatr

```
usage: repeatr [<flags>] <command> [<args> ...]

Functional computation.

Flags:
  -h, --help         Show context-sensitive help (also try --help-long and --help-man).
      --format=ansi  Output api format

Commands:
  help [<command>...]
    Show help.

  run [<flags>] <formula>
    Execute a formula.
```
